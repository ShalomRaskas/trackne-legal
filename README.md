# trackne-legal

Public legal pages for the Trackne iOS/Android app — privacy policy, terms of service, and support contact.

Hosted on GitHub Pages.

## Pages

- `index.html` — landing with links
- `privacy.html` — privacy policy
- `terms.html` — terms of service
- `style.css` — shared clinical stylesheet (paper + ink + muted medical blue, Newsreader serif)

## Updating

Edit the HTML, commit, push. GitHub Pages redeploys automatically within ~30 seconds.
